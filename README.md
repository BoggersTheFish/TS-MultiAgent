# TS-MultiAgent

**Multi-agent coordination for the Thinking System (TS)**

Agents as nodes in a shared tension graph. Waves propagate across agent boundaries → emergent negotiation, alignment, and collective hypothesis formation. No central controller — pure substrate-level coordination.

## Quick Start
```bash
git clone https://github.com/BoggersTheFish/TS-MultiAgent.git
cd TS-MultiAgent
pip install -r requirements.txt
python -m ts_multiagent.demo
```

## Core Concepts
- **Agent Nodes**: Independent TS instances with local graphs
- **Inter-Agent Edges**: Tension-weighted constraints between agents
- **Shared Wave Propagation**: Activation spreads + tension resolution across the swarm
- **Emergent Behaviors**: Consensus, specialization, conflict resolution via wave interference

Fits perfectly into Wave 16: Multi-Agent Coordination.
